우선 조치 필요 항목

구분 상태 내용 권장 조치
보안 경고 public.set_updated_at 함수에 고정 search_path 설정이 없습니다. 함수 재정의 시 set search_path = public 또는 더 엄격한 경로를 명시합니다.
성능 개선 필요 public.reports.created_by 외래키에 covering index가 없습니다. create index if not exists idx_reports_created_by on public.reports(created_by);
성능 개선 필요 public.report_events.user_id 외래키에 covering index가 없습니다. create index if not exists idx_report_events_user_id on public.report_events(user_id);
